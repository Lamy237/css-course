## Introduction

[CSS Media Queries](https://developer.mozilla.org/en-US/docs/Learn/CSS/CSS_layout/Media_queries) allow you to modify your site based on specific characteristics and 
parameters.

### Syntax
```css
@media media-type and (media-feature-rule) {
  /* CSS rules go here */
}
```
